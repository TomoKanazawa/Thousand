# Thousand Health — Brand Assets

## Selected logo

**`logo-thousand-1000.svg`** — "1000" set in **Hoefler Text Black**, arranged as an
aligned 2×2 grid (1 0 / 0 0). Vector, transparent background, fill `#0a1722`.

- Served by the web app at `web/public/logo-1000.svg`.
- App icons / favicons are generated from this file (see below).

## Generated icons (in `web/public/`)

White background, ~20% rounded corners, logo at ~66% with margin. Rendered at 4×
and downscaled for crisp anti-aliasing.

| File | Size | Use |
|---|---|---|
| `favicon.ico` (in `web/app/`) | 16–256 multi-res | Browser tab (auto-served by Next at `/favicon.ico`) |
| `favicon-16x16.png` … `favicon-128x128.png` | 16/32/48/64/96/128 | Browser / bookmarks |
| `apple-touch-icon.png` | 180 | iOS home screen |
| `icon-192.png`, `icon-256.png`, `icon-512.png` | 192/256/512 | PWA / Android |
| `icon-1024.png` | 1024 | Share / store listing |
